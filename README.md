# Анализ продаж Superstore

## Цель проекта
Построить дашборд для руководителя с ключевыми метриками и выводами по продажам.

## Инструменты
- **Tableau Public** — для визуализации и создания дашборда
- **Датасет**: Superstore Sales (Kaggle)

## Ключевые метрики
- Общая выручка: 2 297 201 ₽
- Количество заказов: 9 994
- Средний чек: 229,9 ₽

## На дашборде представлены
- Выручка по категориям товаров
- Динамика продаж по месяцам
- Интерактивный фильтр по категориям и сегментам

## Выводы
- Самая прибыльная категория — Техника
- Пик продаж приходится на ноябрь, спад в феврале
